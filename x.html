隐私政策 – 阴阳回复助手

更新日期：2026-01-27

阴阳回复助手是一款帮助用户生成辛辣回复内容的浏览器扩展。我们非常重视您的隐私，因此做出以下声明：

本扩展不收集、存储、共享或出售任何用户的个人信息。
本扩展不会上传您的浏览内容或生成的回复到我们的服务器。
使用的 "storage" 权限仅用于保存您的 API Key 和攻击模式设置。这些数据仅存储在本地浏览器中，不会上传到我们的服务器。
当您使用 DeepSeek API 功能时，您的请求内容会发送到 DeepSeek API 服务器进行处理。请参阅 DeepSeek 的隐私政策了解其数据处理方式。
本扩展不会跟踪、分析或记录您的浏览行为。
我们不与任何第三方服务共享、出售或交换数据。
您可以随时从浏览器中卸载本扩展，所有本地数据将随即删除。
